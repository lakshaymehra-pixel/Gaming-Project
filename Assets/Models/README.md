# Character models

`Swat.fbx` — from the **Ultimate Modular Men Pack** by [Quaternius](https://quaternius.com/),
downloaded via [Poly Pizza](https://poly.pizza/m/Btfn3G5Xv4).

License: **CC0 (public domain)** — free for personal and commercial use, no attribution
required. Credited anyway, because the pack is excellent.

The FBX carries its own rig (`CharacterArmature`) and a full animation set embedded in the
file: Idle, Idle_Gun, Idle_Gun_Shoot, Gun_Shoot, Run (all directions), Walk, Death,
HitRecieve, and more. `SoldierFactory` builds the enemy from this model and falls back to
the primitive soldier if the file is missing.
